프로젝트 구조 최초 설정 실행 방법 엑셀 시트 컬럼 설명 test_type locator 규칙 실전 시나리오 결과 확인 오류 해결

3. 실행 방법

# 대화형 (프롬프트로 URL → 브라우저 → 시트 순서로 입력)
python main.py

# URL 직접 지정
python main.py -u https://example.com

# 브라우저 + 시트 지정
python main.py -u https://example.com -b chrome -s test_suite

# 통합 스위트 실행 (test_suite 시트 자동 선택)
python main.py -u https://example.com --run-suite

# Firefox로 실행
python main.py -u https://example.com -b firefox

# 시트 목록 확인
python main.py --list-sheets

# 샘플 엑셀 재생성
python main.py --init

CLI 옵션 전체

옵션기본값설명
-u없음테스트 대상 URL (필수)
-bchrome브라우저 (chrome / firefox / edge / safari)
-stests엑셀 시트 이름
-fdata/test_data.xlsx외부 엑셀 파일 경로
--run-suite-test_suite 시트 전체 실행
--init-샘플 엑셀 생성 후 종료
--list-sheets-엑셀 시트 목록 출력 후 종료

4. 엑셀 시트 구성

시트명용도권장
tests페이지 로딩 확인
click_tests클릭 테스트
form_tests입력/제출 테스트
test_suite모든 타입 통합 (한 시트에서 전체 시나리오 실행)권장
test_suite 시트를 사용하면 페이지 로딩 → 클릭 → 입력 → 텍스트 확인을 한 시트에서 순서대로 실행할 수 있습니다.

5. test_suite 시트 — 전체 컬럼 설명

컬럼명필수설명
test_id필수테스트 케이스 ID (예: TC_001)
test_type필수실행할 테스트 타입 (아래 test_type 표 참고)
description권장테스트 설명 (리포트에 표시)
클릭 대상 엘리먼트
locator_type조건부id / class / css / xpath / name / partial_link_text / tag
locator_value조건부엘리먼트 식별값
nth선택n번째 엘리먼트 (1부터, 비워두면 첫 번째)
click_and_verify 전용 — 클릭 후 확인할 엘리먼트
verify_locator_type조건부확인 대상 locator_type
verify_locator_value조건부확인 대상 locator_value
verify_nth선택확인 대상 nth
verify_expect조건부present(기본) / absent / text
input / input_and_submit 전용
input_text조건부입력할 텍스트
input_locator_type조건부입력 필드 locator_type
input_locator_value조건부입력 필드 locator_value
input_nth선택입력 필드 nth
submit_locator_type조건부제출 버튼 locator_type
submit_locator_value조건부제출 버튼 locator_value
submit_nth선택제출 버튼 nth
검증 전용
expected_text조건부텍스트 비교 예상값 (verify_text / click_and_verify 텍스트 모드)
expected_url_contains선택URL에 포함되어야 할 문자열
select 전용

select_value조건부선택할 옵션값
대기 옵션
wait_before선택액션 전 대기(초) — 페이지 로딩 후 추가 대기 시
wait_after선택클릭 후 검증 전 대기(초) — 팝업/애니메이션 대기 시

7. locator_type 입력 규칙

locator_typelocator_value 예시주의사항
idlogin-btn# 없이 id명만 입력
classbtn-submit. 없이 class명만 입력. 공백 포함 시 자동으로 CSS 변환
cssh2.subtitle. # 포함하여 입력
cssinput[type='text']속성 선택자 가능
cssh2.class-a.class-b클래스 두 개 동시 지정
xpath//h2[@class='subtitle']XPath 전체 표현식 입력
xpath//h2[contains(@class,'sub')]부분 클래스 포함 검색
nameusernamename 속성값
partial_link_text로그인링크 텍스트 일부
tagh1태그명만 입력

nth 사용 규칙

nth 값의미
비워두기 또는 1첫 번째 엘리먼트 (기본값)
2두 번째 엘리먼트
3세 번째 엘리먼트

10. 자주 발생하는 오류

증상원인해결
ModuleNotFoundError 패키지 미설치 pip install -r requirements.txt
FileNotFoundError 엑셀 파일 없음 python main.py --init
FAIL — 엘리먼트 못 찾음 locator_value 오타 또는 잘못된 locator_type 브라우저 개발자도구(F12) → 해당 엘리먼트 우클릭 → 검사로 확인
FAIL — 텍스트 불일치 공백·줄바꿈 차이 리포트 에러 메시지의 실제값을 복사해서 expected_text에 붙여넣기
FAIL — 텍스트가 빈 값 Vue/React 동적 렌더링 wait_before / wait_after 값 늘리기 (1~2초). 자동으로 JS textContent 재시도함
팝업 확인 FAIL 팝업 뜨는 시간 부족 wait_after 값 늘리기 (1~2초)
클릭이 안 됨 화면 밖 엘리먼트 또는 overlay 자동으로 hover → scroll → JS클릭 3단계 시도. wait_before 추가
WebDriverException 브라우저 드라이버 문제 인터넷 연결 확인 (webdriver-manager가 자동 다운로드)
